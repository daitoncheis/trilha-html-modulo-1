História da Web

A história da Internet é também acompanhada da história da técnologia como um todo. Desde de o surgimento dos primeiros computadores, como por exemplo o Harvard Mark I, o Colossus e o ENIAC que eram máquinas enormes e que funcionavam com válvulas, até os computadores atuais que são ultrafinos e portáteis com uma quantidade enorme de chips de circuitos integrados.

A Internet que surgiu em um primeiro momento para a comunicação entre universidades e em seguida foi utilizada no ambiente militar, logo foi percebida como uma ferramenta que poderia ser amplamente utilizada, sendo difundidade em todas as áreas de atuação.

O que são Clients?

Nesta etapa foram apresentados os conceitos de Client, Navegadores, Aplicações Web e Dispositivos móveis.

Client é o termo utilizado para o lado do usuário em uma aplicação ou programa, que recebe os dados do servidor para utiliza-los em sua máquina local.

Navegadores são programas utilizados para a intepretação das páginas HTML recebidas do servidor. Há uma grande variedade de navegadores, também conhecidos como "browsers", porém a intepretação do arquivo pode variar de um para o outro dependendo da tecnologia utilizada.

Aplicações Web são soluções que utilizam a internet como meio de comunição, não sendo necessária a instalação de qualquer programa específico para a utilização da mesma.

Dispositivos móveis são os equipamentos que podem se conectar a internet e serem utilizados de praticamente qualquer lugar, como por exemplo, laptops, tablets, smartwatchs e smartphones.

O que são Servers?

Servers que significa Servidores, são basicamente computadores mais potentes e com mais performance, onde os Clients (usuários) solicitam ou executam algum tipo de aplicação.

Existem vários tipos de servidores e alguns dos exemplos apresentados foram:

Servidor de arquivos
É um servidor onde é possível o armazenamento de arquivos, um exemplo muito popular é o Google Drive.
Servidor Firewall (Segurança)
É um servidor onde são configuradas as específicações de segurança de uma rede, descrevendo as possibilidades de entrada e saída de pacotes da mesma.
Servidor de Streaming
É um servidor onde as aplicações de Streaming são executadas, um exemplo muito famoso desse tipo de servidor é o Netflix.
Servidor de e-mail
É um servidor onde são armazenados os e-mails enviados e enviados de um domínio ou rede específica. Um exemplo muito famoso é Outlook, muito utilizado por diversas pessoas, além do Gmail da Google.
Servidor Web
É um servidor onde são armazenadas as páginas Web, contendo todos os arquivos que fazem parte da mesma, como os arquivos HTML, CSS e JavaScript.

Linguagens de Programação

Linguagem de programação é uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para a criação de programas (softwares). Cada linguagem de programação tem sua particularidade e usuabilidade. Há linguagens que são utilizadas em servidores (Server-side), como há linguagens utilizados na máquinas de usuários (Client-side)

Linguagens Server-side são linguagens que são executadas em servidores, como por exemplo PHP, Ruby, C#, Java, JavaScript, etc.

Estrutura básica de um HTML

HTML é uma linguagem de programação marcação na qual são utilizados textos delimitados por nomes que o navegador consegue interpretar, cada nome destes delimitadores tem uma função e comportamento específico. Os delimitadores são chamados de Tags, a seguir será observados as particularidades de algumas das Tags mais utilizadas e conhecidas em HTML.

Falando sobre Tags

Tags são palavras-chave que definem como o navegador irá formatar e apresentar o conteúdo. Com a ajuda das Tags, o navegador pode distinguir a diferença entre um conteúdo HTML de um conteúdo comum. Tags são formadas por três partes principais: a abertura, o conteúdo e o fechamento. Porém algumas Tags não necessitam de fechamento. Todas as Tags devem estar entre os símbolos < e >.

Atributos das Tags

Atributos são palavras especiais que promovem informações adicionais ou modificam as Tags. Cada Tag pode ter atributos que definem o comportamento da mesma.

Existem atributos que são considerados genéricos, que podem ser usados em qualquer Tag, como por exemplo: <id> e <title>. Porém existem tags com atributos específicos, como a tag <a>, qual o atributo é "href".

Os nomes e valores dos atributos são "case sensitive", ou seja, diferenciam letras maiúsculas de minúsculas, portanto é recomendado pela W3C que sejam escritos apenas em letras minúsculas.

Tags de Texto

Algumas das Tags mais conhecidas e utilizadas em HTML são as seguintes:

  • <H1> até <H6> tamanho de títulos dos parágrafos
  • <p>quebra de parágrafos
  • <i>itálico
  • <u>sublinhado
  • <strong>negrito
  • <ol>lista ordenada
  • <ul>lista não ordenada
  • <li>item da lista

Listas Ordenadas e Não Ordenadas

Listas ordenadas podem ser ordenadas numéricamente ou em ordem alfabética.

Listas não ordenadas podem ser ordenadas por "bullet points" marcação em pontos.

Em ambas as listas a tag <li> deve ser utilizada para a adição dos itens a lista.
